## Run Locally

Clone the project

```bash
  git clone git@github.com:thinkphp/react-e-commerce.git
```

Go to the project directory

```bash
  cd react-e-commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
